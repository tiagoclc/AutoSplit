Aplicativo para dividir a tela automaticamente e abrir um aplicativo em cada lado da tela, a escolha do usuário.
O adb deve estar ativado.
A simulação do toque na tela, que abre o splitscreen está definido para a posição do botão na tela do dolphin. Se o seu carro tiver o botão em outra posição da tela, deve definir manualmente as coordenadas do input tap.
